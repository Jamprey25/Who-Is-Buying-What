# Who-Is-Buying-What
An AI-driven M&amp;A intelligence platform democratizing real-time deal data. Using Gemini 1.5 Flash and RAG, it parses SEC 8-Ks into structured insights in &lt;120s.  Tech: Next.js, FastAPI, Supabase, AWS. Flow: Ingest (SEC/News) → Extract (LLM) → Enrich (APIs) → Deliver (WebSockets). Goal: High-fidelity, low-latency market tracking.
